Using EduGenAI? Follow these instructions to add this as an extension. Using a different tool? Contact me using one of the links above and I'll add instructions for it.

When using the standalone tool, always confirm with your institution whether privacy policy allows this use. Tool may contain false positives & negatives, and checks against abstracts only. Always confirm manually.

Used only for this analysis, sent directly to the provider, never stored or logged.
OpenAlex works without a key. If you have a Premium key, it is likewise one-time use and never stored.
Caps each model call. Raise it for very long bibliographies; lower it to control cost. 1000–64000.
What data leaves your machine?
  • OpenAlex receives only reference titles and DOIs — never the paper's prose, the student's name, or any body text.
  • Your chosen LLM receives the document text (needed to locate the reference list and citing sentences) and, for the misquote check, the citation snippets plus OpenAlex abstracts — nothing more.
  • All three providers state that data sent via an API key is not used to train their models by default. Since you bring your own key, the request runs under your account's terms.
  • This tool stores nothing: no database, no file writes, no logs of your document or keys.
